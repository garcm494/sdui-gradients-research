# Gradient Configuration

## Phase I. Linear Gradients
Up, Down, Left, Right, Diagonally.

## Requirements:
At least 2 colors.
Starting point.
Direction (or angle).
Gradient type (linear, radial, conic).

### Sample Top to Bottom.

```
{
	"gradient-sample": {
		"colors": ["BE6666", "C16969"],
		"starts": "top",
		"direction": "down",
		"type": "linear"
	}
}
```

### Sample on theme document.
```
{
	"gradients": {
		"hubCards": {
			"colors": ["000000", "FFFFFF"],
			"starts": "top",
			"direction": "down",
			"type": "linear"
		},
		"homeCards": {

```
